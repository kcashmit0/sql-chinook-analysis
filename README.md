#SQL Chinook Database Analysis
Learning SQL JOINs and advanced queries using the Chinook music store database.

## Database Schema
- 11 interconnected tables
- Music store scenario (artists, albums, tracks, customers, invoices)

## Learning Goals
- Master all JOIN types
- Practice aggregations with multiple tables
- Build real business analysis queries

## Progress
- Week 1: Basic JOINs (INNER, LEFT)
- Week 2: Multiple table JOINs
- Week 3: Advanced queries with subqueries
- Week 4: Window functions and CTEs

## Tools
